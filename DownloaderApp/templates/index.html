Youtube Audio Downloader

This conversion tool allows you to download mp3 files from Youtube in bulk. Simply make a text file with names of songs you want to download, each seperated by line. It will perform a search on Youtube to find the top search result, and download using PyTube. It will return a zip file with your songs inside as mp3 format.
